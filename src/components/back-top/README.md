## BackTop 警告

回到顶部

### 基本用法

```js
import BackTop from '@/components/back-top'
```

```html
<back-top :visible-height="200" :rate="6" :custom-style="{background: '#e7eaf1'}"></back-top>
```

### Attributes

| 参数          | 说明       | 类型   | 可选值 | 默认值 |
| ------------- | ---------- | ------ | ------ | ------ |
| visibleHeight | 高度       | number | —      | 400    |
| rate          | 速率       | number | -      | 6      |
| customStyle   | 自定义样式 | object | —      | —      |
